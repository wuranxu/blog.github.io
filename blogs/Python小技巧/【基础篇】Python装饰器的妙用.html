要求实现一个自定义的装饰器

这题的核心内容就是callable，我们可以这么写:

def run(times):
    if callable(times):
        def wrapper(*args, **kwargs):
            for i in range(5):
                times(*args, **kwargs)
                print(f"运行的第{i}次")

        return wrapper
    else:
        def decorator(func):
            def wrapper(*args, **kwargs):
                for i in range(times):
                    func(*args, **kwargs)
                    print(f"运行的第{i}次")

            return wrapper

        return decorator

                print(f"运行的第{i}次")

我们说了那么多，该讲下装饰器的运用了。以下都是需求+例子，大家且看且珍惜~

  • 计时器

    当我们需要统计一些方法的执行时间，我们可以这么写。

def time_dec(func):
    def wrapper(*arg):
        t = time.perf_counter()
        res = func(*arg)
        print(func.__name__, time.perf_counter() - t)
        return res
    return wrapper

  • 异常捕获器（来自测试开发笋货的思路）

    写try和except写厌烦了吧？你可以这么玩:

def exceptions(func):
    def wrapper(*args, **kwargs):
        try:
            return func(*args, **kwargs)
        except Exception as e:
            raise Exception(e)

    return wrapper

def login_required(func):
        @wraps(func)
        def wrapper(*args, **kwargs):
            try:
                headers = request.headers
                token = headers.get('token')

                # 解析用户信息
                user_info = UserToken.parse_token(token)

  • 给变量加锁

import functools

def synchronized(lock):
    """ Synchronization decorator """
    def wrap(f):
        @functools.wraps(f)
        def newFunction(*args, **kw):
            with lock:
                return f(*args, **kw)
        return newFunction
    return wrap
