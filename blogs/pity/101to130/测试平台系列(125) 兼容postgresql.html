所以这意味着，如果我们要兼容pg，则需要把datetime类型全部换成TIMESTAMP，换是可以换，但我就怕我们现在的代码会受影响。

# 适配apscheduler

总体来说，由于apscheduler的存在，我们还是得弄一套同步session，至于pg的同步session，我们需要安装psycopg2:

pip install psycopg2
1
