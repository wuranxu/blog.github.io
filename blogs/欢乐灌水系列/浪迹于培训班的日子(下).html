浪迹于培训班的日子(下).md

浪迹于培训班的日子(下).md

米洛 2022/6/13 杂谈

大家好，这里是老克，一个想和大家一起分享测试开发相关的技术，面试经验和成长经历的博主！

欢迎大家关注我的公众号: 米洛的测开日记。

上篇说到，在完成了第一天的内容后，小克他晚上回去加练，终于累到睡着了。

# 第二天

第二天7点小克便早早起来了，和2个舍友一起，买了个千层饼（其实后来到了上海我才知道那个是酱香饼）和一杯豆浆就边走边吃，开等公交了。车程大约半个多小时吧，我们和往常一样，爬着楼梯到教室。

其实我已经记不得到底我在那个教室待了多久，关于里面的印象很少，所以就粗略算自己坚持了2天吧。因为学的内容虽然基本都已经忘记了，但是我还记得舍友小柳，到下午3点多的时候，躲在最后一排玩三国杀。

对了，说到培训班。其实当时每个班级都会有个助教类的人物，这个人物我记得我们班也有。他是一个胖胖的胖子，徐老师介绍的时候把他吹的神乎其神，说他经验很丰富，当了很多期的助教了。在老师来不及答疑的时候，可以请教助教。其实我这个三国杀的舍友，和助教一起坐在最后一排，所以他很快和助教混熟了。

后来我才知道，所谓的助教，就是前几期没能顺利毕业的学生，跟着一起继续学的，除了偶尔维护下纪律，更多的是二次学习的任务。这是我对助教的印象。

记得当时第二天，老师给我们安排了一些接下来的项目。首先他展示了一个坦克大战的项目，然后说我们毕业后每个人都能做出来这样的项目，我听了很是心动。做游戏是我的毕生梦想，虽然从来没有敢去尝试过。哪怕是这样一个小小的坦克大战，我都会很期待。在我眼里，做游戏远比打游戏更好玩。

但那个是进阶任务啦，当时给我们说的是接下来会先做一个MP3播放器，这对于当时渺小的我来说，感觉是有些不可思议的。也不知道是不是我自己多心，对于我来说，我认为我的C++语法都不是很牢固，前面还在学Linux命令，现在就开始写mp3播放器了？

我不禁对老师的安排有些怀疑，或者说我开始担心自己跟不上。

# 需要贷款

前文提到过，我们仨只付了一个"首付"，对了此时我想起来了，我应该总共上了大概一周，是快到办手续的时候我萌生了退意。

前面一周基本上都是试听课的阶段，过完一周后就得充值了。所以在中午午休的时候，当时招生的老师，找到了我，想让我签个字。条款什么的，我都还没有看，我就看到xxx贷款协议，我瞬间就明白了这事不是那么简单。

培训费不知道是5000还是8000，但是看着贷款单子上面大概要还2万的样子，于是我陷入了沉思。打算和家里人商量商量。

# 做决定

记得当时给我在老家的爸爸打了个电话，我爸表示可以负担我这个学费，说没必要贷款，可以直接给我全款。那时和他关系不太和睦的我，突然觉得挺对不起家里人。觉得自己蹉跎3年，在大学里面还因为旷课被叫过家长。

这里补充一件很忏悔的事情，当时我逃课特别厉害，辅导员有点管不住我，于是他悄悄打电话给我爸了（这是我事先不知道的）。我爸远在荆州，正好在我生日的那天赶来了我的学校，然后给我带了个大蛋糕，还带我和我舍友去下了个馆子。

我以为我爸只是来给我过生日，我爸也没有什么异常表现出来，只是乐呵呵地和大家开着玩笑，其实他是给我面子。

吃完饭，他就打算回家了。我在送他出发的路上，他和我说，其实他来是因为我辅导员给他打了几次电话，说我逃课很严重，想让他劝劝我，说完后他就走了，走之前只叫我好自为之吧，也没有什么多余的话要对我说了。

可能是当时学的不算突出，加上突然想到这件事，觉得很对不起家里人，可能学完了也没啥区别，反而多一笔负担。所以我打算退了这个班，去自学。希望能靠自己，而不是靠我爸赞助我的学费去学习。

于是我给培训班的老师说，我不继续学了，前面的钱也不用退了，我后面贷款就不办了。招生的老师比较体贴人，问了我很多问题，我当时就是不松口，坚持不学了。我说我要自学，他看我态度坚决，就决心不劝我了，让徐老师和我谈谈。

# 最难忘

当时徐老师过来，问我为什么不想继续学了，是不是觉得太难了之类的。可惜他比较狡猾，不管我怎么说，我说我想自学，他就跟我强调自学没有这边学的系统之类的。后来我有点摆烂，说就算自学学不好，也不继续学了。

"....你这种，一点都不能坚持，怎么能混的好。我这个班里的人都是冲着年薪10万去的..."

我闷闷不吱声。

他见劝不动我，说出了让我至今难忘的话:

你以后，可能在某个餐馆，洗着盘子，一个月拿着2 3000块钱的工资，估计你一辈子也就这样了。

他看我惊讶的样子（其实我是没想到他能说出这样的话），他继续说:

这样，以后如果你一个月工资能有5000的话，你可以来找徐老师，我就在这等着你。就这样，好吧，多的徐老师就不多说了。

那一刻，我是真的有些抬不起头来。可能是我一辈子第一次被人这么瞧不起！

我记得我当时月薪到5k我还在想，我要是去找他，他会不会说啥通货膨胀啊，你是在一线城市不一样啊之类的。后来也释怀了，不是他我说不定没那么努力。

小贴士：上篇文章末尾图片洗盘子，就是在说自己如果不努力就真如他所说了。

# 2个舍友的后续

2个舍友C++基础比我差点，我虽然逃课，但是C++班上是考第一的。他们学到后面越来越吃力。我们还是住一起，他们上课我就在家啃<<C++ primer>>。

暑假很快就过去了，课程也几乎到了尾声，他们也到了开始找工作的环节了。但是其实他们也没学到啥内容，毕业以后就开始了还贷生涯。这也是我叫他们倒霉蛋的原因。

后来他们还时不时对我说好后悔，小柳后来在家里人介绍下去了武汉一家日企上班，每个月4 5k的工资还要还2k贷款。经常对我说特别特别后悔，没和我一起走。其实我当时也劝过他们，但是他们不想轻易放弃吧。

另一个舍友，虽然没转行，但是也没能找到C++的工作。

# 说点自己的看法吧

  1. 培训和自学，其实培训好处还是大于自学的，也分机构。我们当时的小培训机构，可能忽悠成分居多。我也有很多同学是去了培训班的，出去以后也慢慢走上了正轨，比如我那个培训Java的同学，后来顺利就业，现在在平安应该。

  2. 培训机构如果不搞花里胡哨的贷款，专注育人，也不夸夸其谈，循序渐进。像饭佬这种水平的人，如果当他的学子，无疑是一种幸运。

  3. 还是要好(四声)学，你好学，自学会厉害，培训会更厉害，只会学多不会学少。培训或者说指导，有个真有水准的人指导，会少走弯路，事半功倍。包括大家经常熟悉的某某魔法学院，我相信它的初心也是为了培育更多优秀的测试工程师。

  4. 我不盲目吹捧培训，也不盲目贬低培训，万物皆有两面性。培训确实帮了很多人入门，但也淘汰了不少人。见过培训后走上正轨的，也见过培训后还是摸不到门路的。人和人是不一样的，这也是为什么培训那么适合转行的朋友，因为它系统教学，比自己盲人摸象好。

总结一下吧，如果去培训，要找个靠谱的。如果不想花钱培训，请给自己定好计划，用兴趣驱动学习。

加油，打工人们！
