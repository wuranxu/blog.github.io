最划算地入手年轻人的第一台云服务器.md

米洛 2022/6/13 实用工具

我是米洛，一个测试开发领域博主。关注米洛的测开日记，解锁更多踩坑姿势。

本文非软文，纯粹是个人经验之谈。给大家讲讲一些云服务器的知识，特别是自己想要部署个自己的站点的朋友。

# 云服务器

云服务器这个概念大家肯定都不陌生，简单的说就是托管式的服务器，给你一台机器的使用权限，假设1年。这1年内，这机器你可以远程访问，可以在上面为所欲为，做你想要做的事情。

# 会不会买了扔着不用

其实买啥东西，都是存在这种可能的。就好比博主17年买了台macbook pro，因为18年公司配的即是mbp，所以毫无用武之地，基本上闲置到了现在。

这玩意还是看自己，假设你正在和 @我去热饭 学习怎么编写测试平台，或者你想写一些实用的工具：比如定时给你的男女朋友发送生日祝福等等。

博主曾经写了个boss直聘过滤消息，自动回复消息的脚本。如果放在自己电脑上跑，可能电脑得一直开着，非常难用。

但是云服务器的好处就是，有专人托管，24小时待命，电费也不需要你出，还有公网ip。

# 厂商对比

因为这块咱不是专业的，所以简短说说。

一般来说在我心中这些厂商的排名是:

A > T > B > HW = 其他

毫无疑问，aliyun是最专业，最可靠的。但其实对我们个人用户来说，基本都差不多，所以我们更在意的是性价比。

# 云服务器地点

很多服务器厂商都会给你服务器的地点，比如同样的配置，香港的（算海外）就比较贵，我买过一台tx云的，400多3年，1核1G的，现在闲置着快要到期了。

大家一般还是买国内的比较好，最好买自己城市附近的，一般是按华东华南华北这样区分地点。离得越近，访问速度自然是越快咯。

至于新加坡日本这样的国外服务器，一般不会出现在优惠活动里，大家也不太需要考虑。

# 购买时机

理性消费，想好自己有没有对应的诉求。

在618和双11的时候，这些厂商的优惠力度是最大的。比如博主现在用的aliyun服务器，就是19年双11购买的。当时价格是220多，3年。平均算起来也就70多块钱一年。配置的话，确实低了点，1核2G的配置，1M的带宽，基本上只能跑很小的程序。

我目前写的平台良好地运行在上面，自己在服务器安装了mysql。目前瓶颈是带宽比较少，1M的带宽显得网站比较慢。

当然那些厂商都是针对了不同用户的，我这个是初级用户，也是最便宜的。比如我当时也有2核4G 2M的配置可选，同样都是3年期限，但那个得400多了。

所以选择也挺重要的，不同的配置有不同的价格。如果对带宽没有要求，比如我某个朋友，喜欢玩儿BTC，会经常关注这块行情。他就有个机器人，天天给他推送价格和每天的收益，这种就不需要多好的配置和带宽。

如果大家只是有个人网站这样的需求，那么1核2G基本上够用了。如果你想玩docker玩k8s玩其他的，那还是需要配置好点的。

划重点: 每年的双11，618最优惠，双12也较为优惠，大家可以对比各大厂商的活动，选择最划算最适合自己的机器。

需要注意的是，这些优惠活动往往针对新用户，但一般人也都是新用户。但大家要注意的是，邀请他人购买服务器是可以赚取佣金的。所以如果你要买服务器，配合老帐号邀请，你就能再打个折扣。比如你同学A是老用户，你让他邀请你购买，你买了以后他会到一笔钱，到时候你们分就好了！

如果不是新用户了怎么办？其实我们可以换个家里人的账号就可以了，服务器和帐号没有很强的关联性，我们技术宅一般需要root+密码+ip就可以了，很少需要做其他的。

# 域名相关

如果你和我一样，想要建立一个个人网站，或者一个能给他人参考的测试平台，那好的域名必不可少。

各大厂商都有自己的域名服务，但我个人用过B和A2家，域名备案方面，B家特别麻烦，建议不要考虑。

一个域名大概多少钱呢?如果你不需要.com, .cn, .net这样的域名，那可谓是相当便宜，还不到一顿饭钱。

想象一下，一顿饭都不到的功夫就可以买1年的域名使用权，对其他想访问你个人网站的用户来说简直体验甚好。

譬如我个人最近写的平台，有部署到云服务器，这样如果有小伙伴想要试用，输入https://pity.fun即可访问到，而不需要记住我都记不住的ip。

10来块钱就可以买1年域名，配合云服务器，简直是如虎添翼。博主前几天一口气买了10年，因为想着续费会很贵，所以干脆买久一点了。

website和fun的后缀，价格差不多，10年不到200，算是很优惠了。

# 域名备案

在我们国内，网站都是需要备案的。建议到自己户口所在地备案，就身份证上的地址。同事备案是需要有云服务器的，这点比较流氓。

阿里云备案比较人性化，基本上按照要求提供资料即可，客服会打电话跟你确认是不是本人备案。

# SSL证书

厂商都会提供免费的ssl证书，如今我的服务也开通了https，每次是1年，由第三方机构颁发，申请很容易。

和收费的比起来就是，一年需要重新申请一次。

# 申请七牛云oss

自己的域名可以去七牛免费申请10G oss空间，后续我会慢慢接入。因为我们有的时候需要测试文件上传这样的接口，那文件的数据从哪里来呢？

# 我们都会统一存储到oss里面，还有后续要做的数据驱动，用户上传的测试数据，比如csv多会放到oss里面，当然我这个是演示项目，如果运用到企业中可能就不能再用这样的免费存储了，需要安全性更高的oss，自研或者阿里云oss/腾讯云cos等等。

今天的内容就讲到这里啦，你心动了吗？奉劝大家，如果不是特别有需要，还是考虑清楚了再购买，切记不要冲动消费。
